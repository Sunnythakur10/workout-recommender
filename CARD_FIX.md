# Card Display Fix - Recommended Plan Name

## Issue
The profile card was always showing the same plan name ("3-day Full Body Beginner") regardless of what plan was actually generated by the AI.

## Root Cause
The profile card (right column) was only displaying the **user's input profile** (goal, equipment, experience, etc.) but was **not showing the recommended plan name** that was selected by the AI model.

The card was displaying:
- ✅ User's fitness goal
- ✅ Available equipment  
- ✅ Experience level
- ✅ Time per day
- ✅ Days per week
- ✅ Liked/Disliked exercises

But was NOT displaying:
- ❌ **Recommended Plan Name** (e.g., "4-day Upper/Lower Hypertrophy", "5-day Cardio + HIIT", etc.)
- ❌ **AI Confidence Score**

## Solution
Added a new section to the profile card that displays:
1. **Recommended Plan Name** - Shows the actual plan selected by the AI
2. **Confidence Score** - Shows how confident the AI is in the recommendation

### Code Changes
```python
# Show recommended plan if one exists
if hasattr(st.session_state, 'current_plan') and hasattr(st.session_state, 'current_plan_id'):
    plan_name = st.session_state.current_plan['meta']['name']
    confidence = st.session_state.current_confidence
    st.markdown("---")
    st.markdown("### 🎯 Recommended Plan")
    st.markdown(f"**{plan_name}**")
    st.markdown(f"*Confidence: {confidence:.1%}*")
```

## Result
Now the card correctly shows:

**📋 Your Profile Summary**
- Goal, Equipment, Experience, Time, Frequency

**🎯 Recommended Plan** ← NEW!
- **Plan Name** (e.g., "5-day Muscle Endurance")
- *Confidence: 85.3%*

**Preferences**
- Liked exercises
- Disliked exercises

## Benefits
1. ✅ Users can now see which plan was actually recommended
2. ✅ Clear distinction between INPUT (profile) and OUTPUT (recommended plan)
3. ✅ Confidence score provides transparency about AI recommendation
4. ✅ Card updates correctly when generating new plans with different parameters
5. ✅ Better user experience with complete information

## Testing
To verify the fix:
1. Set different profile parameters (e.g., change days per week from 3 to 5)
2. Click "Generate Workout Plan"
3. Check the profile card - it should now show the recommended plan name
4. Change parameters again and regenerate
5. Verify the card updates with the new recommended plan

The card will now dynamically show the correct plan name based on the AI's recommendation!
